# Progetto-Principi-IRIS-RECOGNITION
Il progetto sviluppa un sistema di riconoscimento dell’iride ispirato all’algoritmo di Daugman: da immagini oculari estraiamo l’IrisCode tramite segmentazione, normalizzazione e filtri tipo Gabor, confrontando i codici con la distanza di Hamming e valutando le prestazioni con istogrammi e ROC.
